hash函数的特征：
对于md5 ：0-2^64-1
对于sha1 :0-2^128-1
特征：1  输出是有范围的  输入可以无穷
     2  相同的输入会得到相同的输出  不同的输入会得到不同的输出结果
     3  离散性且均匀性



hash表就是我们平常学的，而hash表前提是经过hash函数计算出hash值是多少
对于hash表的时间复杂度详解过程可以参考视频--》O(1)
  扩容代价 O(LogN)
  全部重新hash定位拉链表O（n*logN)
  所以每个O（logN)
  由于N可以变得很大所以O（logN)就会去向于O（1）
